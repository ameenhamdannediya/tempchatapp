# what's this 

its chat site where you get you chat without any login , user can just open a chat with strangers or use it as a temporary chat place. its very easy and fast to setup. its will give you a public link in less than a minute once its is executed and anyone with the access to the link can join and chat. every new link will be unique don't have a stable domain cause it uses free cloudflare hosting to expose the site to internet since it is unique everytime it is executed user can have multiple chat at a time if it is executed multiple times .

its is very basic site , that implements the idea i had in mind and if any feedback or request , feel free to rech out to me .




### How to Run
it is not deployed , but user can downlaod the setupNlaunch.py file to your computer 
```
python3 setupNlaunch.py
```
its should work in any OS
